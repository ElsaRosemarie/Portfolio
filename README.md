# Elsa van Dam — Illustration Portfolio

A minimal illustration portfolio for Elsa van Dam, built with Next.js.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Gallery data and images are generated from `Afbeeldingen portfolio/` when you build.

## Live site

After deployment, enable GitHub Pages once:

1. Open **Settings → Pages** on the repo
2. Set **Source** to **Deploy from a branch**
3. Choose branch **`gh-pages`** and folder **`/ (root)`**
4. Save

The site will be live at:

**https://elsarosemarie.github.io/Portfolio/**

## Pages

- **Home** — hero banner and introduction
- **Work** — masonry gallery with category filters
- **Research** — research projects gallery
- **Workshops** — workshop information
- **About** — biography
- **Contact** — email and social links
